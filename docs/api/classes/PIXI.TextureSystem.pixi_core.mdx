## Methods
<article>

### bind: <span><code>(texture&#58; Texture &#124; BaseTexture, location&#58; number) -&#62; void</code></span>
Bind a texture to a specific location
If you want to unbind something, please use `unbind(texture)` instead of `bind(null, textureLocation)`

#### Parameters:
| Name     | Type                                    | Description         |
| -------- | --------------------------------------- | ------------------- |
| texture  | <code>Texture &#124; BaseTexture</code> | Texture to bind     |
| location | <code>number</code>                     | Location to bind at |
</article>


---------------------------------------

<article>

### contextChange: <span><code>() -&#62; void</code></span>
Sets up the renderer context and necessary buffers.

</article>


---------------------------------------

<article>

### ensureSamplerType: <span><code>(maxTextures&#58; number) -&#62; void</code></span>
Ensures that current boundTextures all have FLOAT sampler type, see [PIXI.SAMPLER_TYPES](../enums/PIXI.SAMPLER_TYPES.pixi_constants) for explanation.

#### Parameters:
| Name        | Type                | Description                  |
| ----------- | ------------------- | ---------------------------- |
| maxTextures | <code>number</code> | number of locations to check |
</article>


---------------------------------------

<article>

### reset: <span><code>() -&#62; void</code></span>
Resets texture location and bound textures Actual `bind(null, i)` calls will be performed at next `unbind()` call

</article>
