The Circle object is used to help draw graphics and can also be used to specify a hit area for displayObjects.



        

<article>

### Constructor:
<h4><code>new PIXI.Circle(x&#58; number, y&#58; number, radius&#58; number) </code></h4>

#### Parameters:
| Name   | Type                | Default        | Description                                   |
| ------ | ------------------- | -------------- | --------------------------------------------- |
| x      | <code>number</code> | <code>0</code> | The X coordinate of the center of this circle |
| y      | <code>number</code> | <code>0</code> | The Y coordinate of the center of this circle |
| radius | <code>number</code> | <code>0</code> | The radius of the circle                      |

</article>
