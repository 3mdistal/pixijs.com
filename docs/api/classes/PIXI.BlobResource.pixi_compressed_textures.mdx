Resource that fetches texture data over the network and stores it in a buffer.



        

<article>

### Constructor:
<h4><code>new PIXI.BlobResource(source&#58; string, options&#58; PIXI.IBlobOptions) </code></h4>

#### Parameters:
| Name             | Type                           | Default       | Description                                                                                                                                            |
| ---------------- | ------------------------------ | ------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| source           | <code>string</code>            | <code></code> | the URL of the texture file                                                                                                                            |
| options          | <code>PIXI.IBlobOptions</code> | <code></code> |                                                                                                                                                        |
| options.autoLoad | <code>boolean</code>           | <code></code> | whether to fetch the data immediately; you can fetch it later via [PIXI.BlobResource#load](../classes/PIXI.BlobResource.pixi_compressed_textures#load) |
| options.width    | <code>boolean</code>           | <code></code> | the width in pixels.                                                                                                                                   |
| options.height   | <code>boolean</code>           | <code></code> | the height in pixels.                                                                                                                                  |

</article>
