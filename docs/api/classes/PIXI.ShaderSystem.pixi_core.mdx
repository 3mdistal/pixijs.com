## Methods
<article>

### bind: <span><code>(shader&#58; Shader, dontSync&#58; boolean) -&#62; GLProgram</code></span>
Changes the current shader to the one given in parameter.

#### Parameters:
| Name     | Type                 | Description                                                 |
| -------- | -------------------- | ----------------------------------------------------------- |
| shader   | <code>Shader</code>  | the new shader                                              |
| dontSync | <code>boolean</code> | false if the shader should automatically sync its uniforms. |
#### Returns:
| Type                   | Description                               |
| ---------------------- | ----------------------------------------- |
| <code>GLProgram</code> | the glProgram that belongs to the shader. |
</article>
