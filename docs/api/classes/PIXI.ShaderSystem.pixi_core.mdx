## Methods
<article>

### bind 
**Type:** <code>(shader&#58; Shader, dontSync&#58; boolean) -&#62; GLProgram</code>

**Description:** Changes the current shader to the one given in parameter.

#### Parameters:
| Name     | Type                 | Description                                                 |
| -------- | -------------------- | ----------------------------------------------------------- |
| shader   | <code>Shader</code>  | the new shader                                              |
| dontSync | <code>boolean</code> | false if the shader should automatically sync its uniforms. |
#### Returns:
| Type                   | Description                               |
| ---------------------- | ----------------------------------------- |
| <code>GLProgram</code> | the glProgram that belongs to the shader. |
</article>


---------------------------------------

<article>

### destroy 
**Type:** <code>() -&#62; void</code>

**Description:** Destroys this System and removes all its textures.

</article>


---------------------------------------

<article>

### disposeShader 
**Type:** <code>(shader&#58; Shader) -&#62; void</code>

**Description:** Disposes shader. If disposing one equals with current shader, set current as null.

#### Parameters:
| Name   | Type                | Description   |
| ------ | ------------------- | ------------- |
| shader | <code>Shader</code> | Shader object |
</article>


---------------------------------------

<article>

### generateProgram 
**Type:** <code>(shader&#58; Shader) -&#62; GLProgram</code>

**Description:** Generates a glProgram version of the Shader provided.

#### Parameters:
| Name   | Type                | Description                                     |
| ------ | ------------------- | ----------------------------------------------- |
| shader | <code>Shader</code> | The shader that the glProgram will be based on. |
#### Returns:
| Type                   | Description            |
| ---------------------- | ---------------------- |
| <code>GLProgram</code> | A shiny new glProgram! |
</article>

### getGlProgram

This can be handy for when you to have a little more control over the setting of your uniforms.

#### Returns:
| Type                   | Description                                                   |
| ---------------------- | ------------------------------------------------------------- |
| <code>GLProgram</code> | The glProgram for the currently bound Shader for this context |
</article>

### createSyncBufferGroup 
**Type:** <code>(group&#58; UniformGroup, glProgram&#58; GLProgram, name&#58; string) -&#62; UniformsSyncCallback</code>

**Description:** Will create a function that uploads a uniform buffer using the STD140 standard. The upload function will then be cached for future calls If a group is manually managed, then a simple upload function is generated

#### Parameters:
| Name      | Type                      | Description                                               |
| --------- | ------------------------- | --------------------------------------------------------- |
| group     | <code>UniformGroup</code> | the uniform buffer group to sync                          |
| glProgram | <code>GLProgram</code>    | the gl program to attach the uniform bindings to          |
| name      | <code>string</code>       | the name of the uniform buffer (must exist on the shader) |
#### Returns:
| Type                              | Description |
| --------------------------------- | ----------- |
| <code>UniformsSyncCallback</code> |             |
