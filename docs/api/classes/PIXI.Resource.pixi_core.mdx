<div class="package-name">

[@pixi/core](../modules/pixi_core).Resource

</div>

Base resource class for textures that manages validation and uploading, depending on its type.
Uploading of a base texture to the GPU is required.



        

<article>

### Constructor:
<h4><code>new PIXI.Resource(width&#58; number, height&#58; number) </code></h4>

#### Parameters:
| Name   | Type                | Default        | Description            |
| ------ | ------------------- | -------------- | ---------------------- |
| width  | <code>number</code> | <code>0</code> | Width of the resource  |
| height | <code>number</code> | <code>0</code> | Height of the resource |

</article>
