The Ellipse object is used to help draw graphics and can also be used to specify a hit area for displayObjects.



        

<article>

### Constructor:
<h4><code>new PIXI.Ellipse(x&#58; number, y&#58; number, halfWidth&#58; number, halfHeight&#58; number) </code></h4>

#### Parameters:
| Name       | Type                | Default        | Description                                    |
| ---------- | ------------------- | -------------- | ---------------------------------------------- |
| x          | <code>number</code> | <code>0</code> | The X coordinate of the center of this ellipse |
| y          | <code>number</code> | <code>0</code> | The Y coordinate of the center of this ellipse |
| halfWidth  | <code>number</code> | <code>0</code> | The half width of this ellipse                 |
| halfHeight | <code>number</code> | <code>0</code> | The half height of this ellipse                |

</article>
