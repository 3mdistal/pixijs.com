Resource type for [HTMLVideoElement](https&#58;//developer.mozilla.org/en-US/docs/Web/API/HTMLVideoElement).



        

<article>

### Constructor:
<h4><code>new PIXI.VideoResource(source&#58; HTMLVideoElement &#124; object &#124; string &#124; Array&#60;string &#124; object&#62;, options&#58; object) </code></h4>

#### Parameters:
| Name                | Type                                                                                                 | Default           | Description                                                                                         |
| ------------------- | ---------------------------------------------------------------------------------------------------- | ----------------- | --------------------------------------------------------------------------------------------------- |
| source              | <code>HTMLVideoElement &#124; object &#124; string &#124; Array&#60;string &#124; object&#62;</code> | <code></code>     | Video element to use.                                                                               |
| options             | <code>object</code>                                                                                  | <code></code>     | Options to use                                                                                      |
| options.autoLoad    | <code>boolean</code>                                                                                 | <code>true</code> | Start loading the video immediately                                                                 |
| options.autoPlay    | <code>boolean</code>                                                                                 | <code>true</code> | Start playing video immediately                                                                     |
| options.updateFPS   | <code>number</code>                                                                                  | <code>0</code>    | How many times a second to update the texture from the video. Leave at 0 to update at every render. |
| options.crossorigin | <code>boolean</code>                                                                                 | <code>true</code> | Load image using cross origin                                                                       |

</article>
