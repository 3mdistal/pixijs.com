[@pixi/canvas-extract](../modules/pixi_canvas_extract).CanvasExtract

The extract manager provides functionality to export content from the renderers.
An instance of this class is automatically created by default, and can be found at `renderer.extract`

## Methods
<article>

### base64: <span><code>(target&#58; DisplayObject &#124; RenderTexture, format&#58; string, quality&#58; number) -&#62; Promise&#60;string&#62;</code></span>
Will return a base64 encoded string of this target. It works by calling`CanvasExtract.getCanvas` and then running toDataURL on that.

#### Parameters:
| Name    | Type                                            | Description                                                                           |
| ------- | ----------------------------------------------- | ------------------------------------------------------------------------------------- |
| target  | <code>DisplayObject &#124; RenderTexture</code> | A displayObject or renderTexture to convert. If left empty will use the main renderer |
| format  | <code>string</code>                             | Image format, e.g. "image/jpeg" or "image/webp".                                      |
| quality | <code>number</code>                             | JPEG or Webp compression from 0 to 1\. Default is 0.92.                               |
#### Returns:
| Type                                 | Description                             |
| ------------------------------------ | --------------------------------------- |
| <code>Promise&#60;string&#62;</code> | A base64 encoded string of the texture. |
</article>

### pixels: <span><code>(target&#58; DisplayObject &#124; RenderTexture, frame&#58; Rectangle) -&#62; Uint8ClampedArray</code></span>
