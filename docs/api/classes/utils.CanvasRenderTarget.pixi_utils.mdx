Creates a Canvas element of the given size to be used as a target for rendering to.



        

<article>

### Constructor:
<h4><code>new PIXI.utils.CanvasRenderTarget(width&#58; number, height&#58; number, resolution&#58; number) </code></h4>

#### Parameters:
| Name       | Type                | Default                               | Description                                       |
| ---------- | ------------------- | ------------------------------------- | ------------------------------------------------- |
| width      | <code>number</code> | <code></code>                         | the width for the newly created canvas            |
| height     | <code>number</code> | <code></code>                         | the height for the newly created canvas           |
| resolution | <code>number</code> | <code>PIXI.settings.RESOLUTION</code> | The resolution / device pixel ratio of the canvas |

</article>
