[@pixi/core](../modules/pixi_core).ViewSystem

</div>


The view system manages the main canvas that is attached to the DOM. This main role is to deal with how the holding the view reference and dealing with how it is resized.



        

<article>
### Contructor:
<h4><code>new PIXI()</code></h4>

</article>
            



## Members
<article>

### autoDensity 
**Type:** <code>boolean</code>

**Description:** Whether CSS dimensions of canvas view should be resized to screen dimensions automatically.

</article>


---------------------------------------

<article>

### element 
**Type:** <code>PIXI.ICanvas</code>

**Description:** The canvas element that everything is drawn to.

</article>


---------------------------------------

<article>

### resolution 
**Type:** <code>number</code>

**Default Value**: <code>PIXI.settings.RESOLUTION</code>

**Description:** The resolution / device pixel ratio of the renderer.

</article>


---------------------------------------

<article>

### screen 
**Type:** <code>PIXI.Rectangle</code>

**Description:** Measurements of the screen. (0, 0, screenWidth, screenHeight).
Its safe to use as filterArea or hitArea for the whole stage.

</article>


---------------------------------------


## Methods
<article>

### destroy 
**Type:** <code>(removeView&#58; boolean) -&#62; void</code>

**Description:** Destroys this System and optionally removes the canvas from the dom.

#### Parameters:
| Name       | Type                 | Description                                |
| ---------- | -------------------- | ------------------------------------------ |
| removeView | <code>boolean</code> | Whether to remove the canvas from the DOM. |
</article>


---------------------------------------

<article>

### init 
**Type:** <code>(options&#58; PIXI.ViewOptions) -&#62; void</code>

**Description:** initiates the view system

#### Parameters:
| Name    | Type                          | Description              |
| ------- | ----------------------------- | ------------------------ |
| options | <code>PIXI.ViewOptions</code> | the options for the view |
</article>


---------------------------------------

<article>

### resizeView 
**Type:** <code>(desiredScreenWidth&#58; number, desiredScreenHeight&#58; number) -&#62; void</code>

**Description:** Resizes the screen and canvas to the specified dimensions.

#### Parameters:
| Name                | Type                | Description                   |
| ------------------- | ------------------- | ----------------------------- |
| desiredScreenWidth  | <code>number</code> | The new width of the screen.  |
| desiredScreenHeight | <code>number</code> | The new height of the screen. |
</article>
