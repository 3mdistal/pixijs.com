[@pixi/math](../modules/pixi_math).Rectangle

Rectangle object is an area defined by its position, as indicated by its top-left corner point (x, y) and by its width and its height.



        

<article>

### Constructor:
<h4><code>new PIXI.Rectangle(x&#58; string &#124; number, y&#58; string &#124; number, width&#58; string &#124; number, height&#58; string &#124; number) </code></h4>

#### Parameters:
| Name   | Type                              | Default        | Description                                                |
| ------ | --------------------------------- | -------------- | ---------------------------------------------------------- |
| x      | <code>string &#124; number</code> | <code>0</code> | The X coordinate of the upper-left corner of the rectangle |
| y      | <code>string &#124; number</code> | <code>0</code> | The Y coordinate of the upper-left corner of the rectangle |
| width  | <code>string &#124; number</code> | <code>0</code> | The overall width of the rectangle                         |
| height | <code>string &#124; number</code> | <code>0</code> | The overall height of the rectangle                        |

</article>

### containsRect 
**Type:** <code>(other&#58; Rectangle) -&#62; boolean</code>

**Description:** Determines whether the `other` Rectangle is contained within `this` Rectangle object. Rectangles that occupy the same space are considered to be containing each other. Rectangles without area (width or height equal to zero) can't contain anything, not even other arealess rectangles.

_Note&#58; Only available with **&#64;pixi/math-extras**._

#### Parameters:
| Name  | Type                   | Description                         |
| ----- | ---------------------- | ----------------------------------- |
| other | <code>Rectangle</code> | The Rectangle to fit inside `this`. |
#### Returns:
| Type                 | Description                                                                |
| -------------------- | -------------------------------------------------------------------------- |
| <code>boolean</code> | A value of `true` if `this` Rectangle contains `other`; otherwise `false`. |
</article>

### intersection 
**Type:** <code>(other&#58; Rectangle, outRect&#58; Rectangle) -&#62; Rectangle</code>

**Description:** If the area of the intersection between the Rectangles `other` and `this` is not zero, returns the area of intersection as a Rectangle object. Otherwise, return an empty Rectangle with its properties set to zero. Rectangles without area (width or height equal to zero) can't intersect or be intersected and will always return an empty rectangle with its properties set to zero.

_Note&#58; Only available with **&#64;pixi/math-extras**._

#### Parameters:
| Name    | Type                   | Description                                                                                       |
| ------- | ---------------------- | ------------------------------------------------------------------------------------------------- |
| other   | <code>Rectangle</code> | The Rectangle to intersect with `this`.                                                           |
| outRect | <code>Rectangle</code> | A Rectangle object in which to store the value, optional (otherwise will create a new Rectangle). |
#### Returns:
| Type                   | Description                             |
| ---------------------- | --------------------------------------- |
| <code>Rectangle</code> | The intersection of `this` and `other`. |
</article>


---------------------------------------

<article>

### intersects 
**Type:** <code>(other&#58; Rectangle, transform&#58; Matrix) -&#62; boolean</code>

**Description:** Determines whether the `other` Rectangle transformed by `transform` intersects with `this` Rectangle object. Returns true only if the area of the intersection is &#62;0, this means that Rectangles sharing a side are not overlapping. Another side effect is that an arealess rectangle (width or height equal to zero) can't intersect any other rectangle.

#### Parameters:
| Name      | Type                   | Description                             |
| --------- | ---------------------- | --------------------------------------- |
| other     | <code>Rectangle</code> | The Rectangle to intersect with `this`. |
| transform | <code>Matrix</code>    | The transformation matrix of `other`.   |
#### Returns:
| Type                 | Description                                                                                       |
| -------------------- | ------------------------------------------------------------------------------------------------- |
| <code>boolean</code> | A value of `true` if the transformed `other` Rectangle intersects with `this`; otherwise `false`. |
</article>
