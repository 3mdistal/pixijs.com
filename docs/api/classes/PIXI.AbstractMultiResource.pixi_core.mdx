[@pixi/core](../modules/pixi_core).AbstractMultiResource

Resource that can manage several resource (items) inside. All resources need to have the same pixel size. Parent class for CubeResource and ArrayResource



        

<article>

### Constructor:
<h4><code>new PIXI.AbstractMultiResource(length&#58; number, options&#58; ISize) </code></h4>

#### Parameters:
| Name           | Type                | Default       | Description                         |
| -------------- | ------------------- | ------------- | ----------------------------------- |
| length         | <code>number</code> | <code></code> |                                     |
| options        | <code>ISize</code>  | <code></code> | Options to for Resource constructor |
| options.width  | <code>number</code> | <code></code> | Width of the resource               |
| options.height | <code>number</code> | <code></code> | Height of the resource              |
