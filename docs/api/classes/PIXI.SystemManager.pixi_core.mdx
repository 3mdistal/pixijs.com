### emitWithCustomOptions 
**Type:** <code>(runner&#58; Runner, options&#58; Record&#60;string, unknown&#62;) -&#62; void</code>

**Description:** A function that will run a runner and call the runners function but pass in different options to each system based on there name.

E.g. If you have two systems added called `systemA` and `systemB` you could call do the following&#58;

```
system.emitWithCustomOptions(init, {
    systemA: {...optionsForA},
    systemB: {...optionsForB},
});

```

`init` would be called on system A passing `optionsForA` and on system B passing `optionsForB`.

#### Parameters:
| Name    | Type                                         | Description                       |
| ------- | -------------------------------------------- | --------------------------------- |
| runner  | <code>Runner</code>                          | the runner to target              |
| options | <code>Record&#60;string, unknown&#62;</code> | key value options for each system |
</article>
