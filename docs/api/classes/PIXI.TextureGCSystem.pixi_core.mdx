<div class="package-name">

[@pixi/core](../modules/pixi_core).TextureGCSystem

</div>


System plugin to the renderer to manage texture garbage collection on the GPU, ensuring that it does not get clogged up with textures that are no longer being used.



        

<article>

### Constructor:
<h4><code>new PIXI.TextureGCSystem(renderer&#58; Renderer) </code></h4>

#### Parameters:
| Name     | Type                  | Default       | Description                         |
| -------- | --------------------- | ------------- | ----------------------------------- |
| renderer | <code>Renderer</code> | <code></code> | The renderer this System works for. |

</article>

---------------------------------------

## Methods
<article>

### run 
**Type:** <code>() -&#62; void</code>

**Description:** Checks to see when the last time a texture was used. If the texture has not been used for a specified amount of time, it will be removed from the GPU.

</article>


---------------------------------------

<article>

### unload 
**Type:** <code>(displayObject&#58; PIXI.DisplayObject) -&#62; void</code>

**Description:** Removes all the textures within the specified displayObject and its children from the GPU.

#### Parameters:
| Name          | Type                            | Description                                    |
| ------------- | ------------------------------- | ---------------------------------------------- |
| displayObject | <code>PIXI.DisplayObject</code> | the displayObject to remove the textures from. |
</article>


---------------------------------------

<article>

### postrender 
**Type:** <code>() -&#62; void</code>

**Description:** Checks to see when the last time a texture was used. If the texture has not been used for a specified amount of time, it will be removed from the GPU.

</article>
