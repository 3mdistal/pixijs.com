<div class="package-name">

[@pixi/core](../modules/pixi_core).State

</div>

This is a WebGL state, and is is passed to [PIXI.StateSystem](../classes/PIXI.StateSystem.pixi_core).
Each mesh rendered may require WebGL to be in a different state. For example you may want different blend mode or to enable polygon offsets



        

<article>
### Contructor:
<h4><code>new PIXI()</code></h4>

</article>
            



## Members
<article>

### blend 
**Type:** <code>boolean</code>

**Default Value**: <code>true</code>

**Description:** Activates blending of the computed fragment color values.

</article>


---------------------------------------

<article>

### blendMode 
**Type:** <code>BLEND_MODES</code>

**Default Value**: <code>PIXI.BLEND_MODES.NORMAL</code>

**Description:** The blend mode to be applied when this state is set. Apply a value of `PIXI.BLEND_MODES.NORMAL` to reset the blend mode. Setting this mode to anything other than NO_BLEND will automatically switch blending on.

</article>


---------------------------------------

<article>

### clockwiseFrontFace 
**Type:** <code>boolean</code>

**Default Value**: <code>false</code>

**Description:** Specifies whether or not front or back-facing polygons can be culled.

</article>


---------------------------------------

<article>

### culling 
**Type:** <code>boolean</code>

**Default Value**: <code>false</code>

**Description:** Activates culling of polygons.

</article>


---------------------------------------

<article>

### depthMask 
**Type:** <code>boolean</code>

**Default Value**: <code>true</code>

**Description:** Enables or disables writing to the depth buffer.

</article>


---------------------------------------

<article>

### depthTest 
**Type:** <code>boolean</code>

**Default Value**: <code>false</code>

**Description:** Activates depth comparisons and updates to the depth buffer.

</article>


---------------------------------------

<article>

### offsets 
**Type:** <code>boolean</code>

**Default Value**: <code>false</code>

**Description:** Activates adding an offset to depth values of polygon's fragments

</article>


---------------------------------------

<article>

### polygonOffset 
**Type:** <code>number</code>

**Default Value**: <code>0</code>

**Description:** The polygon offset. Setting this property to anything other than 0 will automatically enable polygon offset fill.

</article>
