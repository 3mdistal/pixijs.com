### from: <span><code>(vertexSrc&#58; string, fragmentSrc&#58; string, name&#58; string) -&#62; Program</code></span>
A short hand function to create a program based of a vertex and fragment shader.
This method will also check to see if there is a cached program.

#### Parameters:
| Name        | Type                | Description                        |
| ----------- | ------------------- | ---------------------------------- |

| name        | <code>string</code> | Name for shader                    |
#### Returns:
| Type                 | Description                        |
| -------------------- | ---------------------------------- |
| <code>Program</code> | A shiny new PixiJS shader program! |
