Resource for a CubeTexture which contains six resources.



        

<article>

### Constructor:
<h4><code>new PIXI.CubeResource(source&#58; Array&#60;string &#124; PIXI.Resource&#62;, options&#58; ICubeResourceOptions) </code></h4>

#### Parameters:
| Name                    | Type                                                    | Default           | Description                                                      |
| ----------------------- | ------------------------------------------------------- | ----------------- | ---------------------------------------------------------------- |
| source                  | <code>Array&#60;string &#124; PIXI.Resource&#62;</code> | <code></code>     | Collection of URLs or resources to use as the sides of the cube. |
| options                 | <code>ICubeResourceOptions</code>                       | <code></code>     | ImageResource options                                            |
| options.width           | <code>number</code>                                     | <code></code>     | Width of resource                                                |
| options.height          | <code>number</code>                                     | <code></code>     | Height of resource                                               |
| options.autoLoad        | <code>number</code>                                     | <code>true</code> | Whether to auto-load resources                                   |
| options.linkBaseTexture | <code>number</code>                                     | <code>true</code> | In case BaseTextures are supplied, whether to copy them or use   |

</article>
