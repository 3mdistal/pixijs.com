## Methods
<article>

### boundArray: <span><code>(texArray&#58; BatchTextureArray, boundTextures&#58; Array&#60;BaseTexture&#62;, batchId&#58; number, maxTextures&#58; number) -&#62; void</code></span>
Assigns batch locations to textures in array based on boundTextures state. All textures in texArray should have `_batchEnabled = _batchId`, and their count should be less than `maxTextures`.

#### Parameters:
| Name          | Type                                    | Description                                            |
| ------------- | --------------------------------------- | ------------------------------------------------------ |
| texArray      | <code>BatchTextureArray</code>          | textures to bound                                      |
| boundTextures | <code>Array&#60;BaseTexture&#62;</code> | current state of bound textures                        |
| batchId       | <code>number</code>                     | marker for _batchEnabled param of textures in texArray |
| maxTextures   | <code>number</code>                     | number of texture locations to manipulate              |
</article>


---------------------------------------
