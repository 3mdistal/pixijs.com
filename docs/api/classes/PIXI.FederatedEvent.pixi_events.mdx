<div class="package-name">

[@pixi/events](../modules/pixi_events).FederatedEvent

</div>


An DOM-compatible synthetic event implementation that is "forwarded" on behalf of an original FederatedEvent or native [Event](https&#58;//dom.spec.whatwg.org/#event).



        

<article>

### Constructor:
<h4><code>new PIXI.FederatedEvent(manager&#58; EventBoundary) </code></h4>

#### Parameters:
| Name    | Type                       | Default       | Description                                                                                                 |
| ------- | -------------------------- | ------------- | ----------------------------------------------------------------------------------------------------------- |
| manager | <code>EventBoundary</code> | <code></code> | The event boundary which manages this event. Propagation can only occur within the boundary's jurisdiction. |

</article>

---------------------------------------
