TimeLimiter limits the number of items handled by a [PIXI.BasePrepare](../classes/PIXI.BasePrepare.pixi_prepare) to a specified number of milliseconds per frame.



        

<article>

### Constructor:
<h4><code>new PIXI.TimeLimiter(maxMilliseconds&#58; number) </code></h4>

#### Parameters:
| Name            | Type                | Default       | Description                                                            |
| --------------- | ------------------- | ------------- | ---------------------------------------------------------------------- |
| maxMilliseconds | <code>number</code> | <code></code> | The maximum milliseconds that can be spent preparing items each frame. |

</article>
