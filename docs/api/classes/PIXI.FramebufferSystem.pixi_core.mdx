<article>

### blit 
**Type:** <code>(framebuffer&#58; Framebuffer, sourcePixels&#58; Rectangle, destPixels&#58; Rectangle) -&#62; void</code>

**Description:** Only works with WebGL2
blits framebuffer to another of the same or bigger size after that target framebuffer is bound

Fails with WebGL warning if blits multisample framebuffer to different size

#### Parameters:
| Name         | Type                     | Description                                                      |
| ------------ | ------------------------ | ---------------------------------------------------------------- |
| framebuffer  | <code>Framebuffer</code> | by default it blits "into itself", from renderBuffer to texture. |
| sourcePixels | <code>Rectangle</code>   | source rectangle in pixels                                       |
| destPixels   | <code>Rectangle</code>   | dest rectangle in pixels, assumed to be the same as sourcePixels |
</article>

<article>

### clear 
**Type:** <code>(r&#58; number, g&#58; number, b&#58; number, a&#58; number, mask&#58; PIXI.BUFFER_BITS) -&#62; void</code>

**Description:** Clear the color of the context

#### Parameters:
| Name | Type                          | Description                                                                                      |
| ---- | ----------------------------- | ------------------------------------------------------------------------------------------------ |
| r    | <code>number</code>           | Red value from 0 to 1                                                                            |
| g    | <code>number</code>           | Green value from 0 to 1                                                                          |
| b    | <code>number</code>           | Blue value from 0 to 1                                                                           |
| a    | <code>number</code>           | Alpha value from 0 to 1                                                                          |
| mask | <code>PIXI.BUFFER_BITS</code> | Bitwise OR of masks that indicate the buffers to be cleared, by default COLOR and DEPTH buffers. |
</article>
