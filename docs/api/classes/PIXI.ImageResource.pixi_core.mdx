Resource type for HTMLImageElement.



        

<article>

### Constructor:
<h4><code>new PIXI.ImageResource(source&#58; HTMLImageElement &#124; string, options&#58; IImageResourceOptions) </code></h4>

#### Parameters:
| Name                 | Type                                        | Default                                        | Description                                           |
| -------------------- | ------------------------------------------- | ---------------------------------------------- | ----------------------------------------------------- |
| source               | <code>HTMLImageElement &#124; string</code> | <code></code>                                  | image source or URL                                   |
| options              | <code>IImageResourceOptions</code>          | <code></code>                                  |                                                       |
| options.autoLoad     | <code>boolean</code>                        | <code>true</code>                              | start loading process                                 |
| options.createBitmap | <code>boolean</code>                        | <code>PIXI.settings.CREATE_IMAGE_BITMAP</code> | whether its required to create a bitmap before upload |
| options.crossorigin  | <code>boolean</code>                        | <code>true</code>                              | Load image using cross origin                         |
| options.alphaMode    | <code>PIXI.ALPHA_MODES</code>               | <code>PIXI.ALPHA_MODES.UNPACK</code>           | Premultiply image alpha in bitmap                     |

</article>
