A Noise effect filter.
original filter&#58; https&#58;//github.com/evanw/glfx.js/blob/master/src/filters/adjust/noise.js



        

<article>

### Constructor:
<h4><code>new PIXI.NoiseFilter(noise&#58; number, seed&#58; number) </code></h4>

#### Parameters:
| Name  | Type                | Default          | Description                                                            |
| ----- | ------------------- | ---------------- | ---------------------------------------------------------------------- |
| noise | <code>number</code> | <code>0.5</code> | The noise intensity, should be a normalized value in the range [0, 1]. |
| seed  | <code>number</code> | <code></code>    | A random seed for the noise generation. Default is `Math.random()`.    |

</article>

<article>

### apply

#### Parameters:
