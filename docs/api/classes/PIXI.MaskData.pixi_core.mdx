Component for masked elements.
Holds mask mode and temporary data about current mask.



        

<article>

### Constructor:
<h4><code>new PIXI.MaskData(maskObject&#58; PIXI.DisplayObject) </code></h4>
Create MaskData

#### Parameters:
| Name       | Type                            | Default           | Description                    |
| ---------- | ------------------------------- | ----------------- | ------------------------------ |
| maskObject | <code>PIXI.DisplayObject</code> | <code>null</code> | object that describes the mask |

</article>

---------------------------------------




## Members
<article>

### autoDetect 
**Type:** <code>boolean</code>

**Default Value**: <code>true</code>

**Description:** Whether we know the mask type beforehand

</article>


---------------------------------------

<article>

### colorMask 
**Type:** <code>COLOR_MASK_BITS</code>

**Description:** Color mask.

</article>


---------------------------------------

<article>

### enabled 
**Type:** <code>boolean</code>

**Description:** If enabled is true the mask is applied, if false it will not.

</article>


---------------------------------------

<article>

### filter 
**Type:** <code>ISpriteMaskFilter</code>

**Default Value**: <code>null</code>

**Description:** The sprite mask filter. If set to `null`, the default sprite mask filter is used.

</article>


---------------------------------------

<article>

### isMaskData 
**Type:** <code>boolean</code>

**Description:** Indicator of the type (always true for [PIXI.MaskData](../classes/PIXI.MaskData.pixi_core) objects)

</article>


---------------------------------------

<article>

### maskObject 
**Type:** <code>PIXI.DisplayObject</code>

**Description:** Which element we use to mask

</article>


---------------------------------------

<article>

### multisample 
**Type:** <code>MSAA_QUALITY</code>

**Default Value**: <code>PIXI.Filter.defaultMultisample</code>

**Description:** Number of samples of the sprite mask filter. If set to `null`, the sample count of the current render target is used.

</article>


---------------------------------------

<article>

### pooled 
**Type:** <code>boolean</code>

**Description:** Whether it belongs to MaskSystem pool

</article>


---------------------------------------

<article>

### resolution 
**Type:** <code>number</code>

**Default Value**: <code>null</code>

**Description:** Resolution of the sprite mask filter. If set to `null` or `0`, the resolution of the current render target is used.

</article>


---------------------------------------

<article>

### type 
**Type:** <code>MASK_TYPES</code>

**Description:** Mask type

</article>
