<article>

### from: <span><code>(originalStyle&#58; TextStyle &#124; Partial&#60;IHTMLTextStyle&#62;) -&#62; HTMLTextStyle</code></span>
Convert a TextStyle to HTMLTextStyle

```js
import {TextStyle } from 'pixi.js';
import {HTMLTextStyle} from '@pixi/text-html';
const style = new TextStyle();
const htmlStyle = HTMLTextStyle.from(style);
```
      

#### Parameters:
| Name          | Type                                                          | Description |
| ------------- | ------------------------------------------------------------- | ----------- |
| originalStyle | <code>TextStyle &#124; Partial&#60;IHTMLTextStyle&#62;</code> |             |
#### Returns:
| Type                       | Description |
| -------------------------- | ----------- |
| <code>HTMLTextStyle</code> |             |
</article>

<article>

### addOverride: <span><code>(value&#58; string) -&#62; void</code></span>
Add a style override, this can be any CSS property it will override any built-in style. This is the property and the value as a string (e.g., `color: red`). This will override any other internal style.

```js
style.addOverride('background-color: red');
```
      

#### Parameters:
| Name  | Type                | Description          |
| ----- | ------------------- | -------------------- |
| value | <code>string</code> | CSS style(s) to add. |
</article>
