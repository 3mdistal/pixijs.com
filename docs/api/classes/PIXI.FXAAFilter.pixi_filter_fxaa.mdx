[@pixi/filter-fxaa](../modules/pixi_filter_fxaa).FXAAFilter


Basic FXAA (Fast Approximate Anti-Aliasing) implementation based on the code on geeks3d.com with the modification that the texture2DLod stuff was removed since it is unsupported by WebGL.

## Members
<article>

### autoFit: <span><code>boolean</code></span>
- **Default Value**: <code>true</code>

If enabled, PixiJS will fit the filter area into boundaries for better performance. Switch it off if it does not work for specific shader.

</article>


---------------------------------------

<article>

### blendMode: <span><code>BLEND_MODES</code></span>
- **Default Value**: <code>PIXI.BLEND_MODES.NORMAL</code>

Sets the blend mode of the filter.

</article>


---------------------------------------

<article>

### enabled: <span><code>boolean</code></span>
If enabled is true the filter is applied, if false it will not.

</article>


---------------------------------------

<article>

### legacy: <span><code>boolean</code></span>
Legacy filters use position and uvs from attributes (set by filter system)

</article>


---------------------------------------

<article>

### multisample: <span><code>MSAA_QUALITY</code></span>
The samples override of the filter instance.

</article>


---------------------------------------

<article>

### padding: <span><code>number</code></span>
The padding of the filter. Some filters require extra space to breath such as a blur. Increasing this will add extra width and height to the bounds of the object that the filter is applied to.

</article>


---------------------------------------

<article>

### program: <span><code>Program</code></span>
Program that the shader uses.

</article>


---------------------------------------

<article>

### resolution: <span><code>number</code></span>
The resolution of the filter. Setting this to be lower will lower the quality but increase the performance of the filter.

</article>


---------------------------------------

<article>

### state: <span><code>State</code></span>
The WebGL state the filter requires to render.

</article>


---------------------------------------

<article>

### uniforms: <span><code>Dict&#60;any&#62;</code></span>
Shader uniform values, shortcut for `uniformGroup.uniforms`.

</article>
