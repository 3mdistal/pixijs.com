Stores a texture's frame in UV coordinates, in which everything lies in the rectangle `[(0,0), (1,0), (1,1), (0,1)]`.

&#124; Corner       &#124; Coordinates &#124;
&#124; ------------ &#124; ----------- &#124;
&#124; Top-Left     &#124; (x0,y0)     &#124;
&#124; Top-Right    &#124; (x1,y1)     &#124;
&#124; Bottom-Right &#124; (x2,y2)     &#124;
&#124; Bottom-Left  &#124; (x3,y3)     &#124;
