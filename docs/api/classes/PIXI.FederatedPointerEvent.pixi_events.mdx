### getLocalPosition 
**Type:** <code>(displayObject&#58; PIXI.DisplayObject, point&#58; PIXI.IPointData, globalPos&#58; PIXI.IPointData) -&#62; P</code>

**Description:** This will return the local coordinates of the specified displayObject for this InteractionData

#### Parameters:
| Name          | Type                            | Description                                                                                                  |
| ------------- | ------------------------------- | ------------------------------------------------------------------------------------------------------------ |
| displayObject | <code>PIXI.DisplayObject</code> | The DisplayObject that you would like the local coords off                                                   |
| point         | <code>PIXI.IPointData</code>    | A Point object in which to store the value, optional (otherwise will create a new point)                     |
| globalPos     | <code>PIXI.IPointData</code>    | A Point object containing your custom global coords, optional (otherwise will use the current global coords) |
#### Returns:
| Type           | Description                                                                                        |
| -------------- | -------------------------------------------------------------------------------------------------- |
| <code>P</code> | * A point containing the coordinates of the InteractionData position relative to the DisplayObject |
</article>
