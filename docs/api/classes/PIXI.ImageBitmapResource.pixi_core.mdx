Resource type for ImageBitmap.



        

<article>

### Constructor:
<h4><code>new PIXI.ImageBitmapResource(source&#58; ImageBitmap &#124; string, options&#58; IImageBitmapResourceOptions) </code></h4>

#### Parameters:
| Name                | Type                                     | Default           | Description                                           |
| ------------------- | ---------------------------------------- | ----------------- | ----------------------------------------------------- |
| source              | <code>ImageBitmap &#124; string</code>   | <code></code>     | ImageBitmap or URL to use                             |
| options             | <code>IImageBitmapResourceOptions</code> | <code></code>     |                                                       |
| options.autoLoad    | <code>boolean</code>                     | <code>true</code> | Start loading process automatically when constructed. |
| options.crossOrigin | <code>boolean</code>                     | <code>true</code> | Load image using cross origin.                        |
| options.alphaMode   | <code>PIXI.ALPHA_MODES</code>            | <code>null</code> | Alpha mode used when creating the ImageBitmap.        |

</article>
