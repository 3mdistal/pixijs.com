[@pixi/filter-blur](../modules/pixi_filter_blur).BlurFilter

The BlurFilter applies a Gaussian blur to an object.
The strength of the blur can be set for the x-axis and y-axis separately.

### new PIXI.BlurFilter: <span><code>(strength&#58; number, quality&#58; number, resolution&#58; , kernelSize&#58; number) </code></span>

#### Parameters:
| Name       | Type                | Default                               | Description                                                         |
| ---------- | ------------------- | ------------------------------------- | ------------------------------------------------------------------- |
| strength   | <code>number</code> | <code>8</code>                        | The strength of the blur filter.                                    |
| quality    | <code>number</code> | <code>4</code>                        | The quality of the blur filter.                                     |
| resolution | <code></code>       | <code>Filter.defaultResolution</code> | The resolution of the blur filter.                                  |
| kernelSize | <code>number</code> | <code>5</code>                        | The kernelSize of the blur filter.Options&#58; 5, 7, 9, 11, 13, 15. |

<article>

### padding: <span><code>number</code></span>
The padding of the filter. Some filters require extra space to breath such as a blur. Increasing this will add extra width and height to the bounds of the object that the filter is applied to.

<article>

### repeatEdgePixels: <span><code>boolean</code></span>

<article>

The resolution of the filter. Setting this to be lower will lower the quality but increase the performance of the filter.
