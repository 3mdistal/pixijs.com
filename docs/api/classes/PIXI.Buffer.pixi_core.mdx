A wrapper for data so that it can be used and uploaded by WebGL



        

<article>

### Constructor:
<h4><code>new PIXI.Buffer(data&#58; PIXI.IArrayBuffer, _static&#58; boolean, index&#58; boolean) </code></h4>

#### Parameters:
| Name    | Type                           | Default            | Description                      |
| ------- | ------------------------------ | ------------------ | -------------------------------- |
| data    | <code>PIXI.IArrayBuffer</code> | <code></code>      | the data to store in the buffer. |
| _static | <code>boolean</code>           | <code>true</code>  | `true` for static buffer         |
| index   | <code>boolean</code>           | <code>false</code> | `true` for index buffer          |

</article>

---------------------------------------




## Members
<article>

### data 
**Type:** <code>PIXI.IArrayBuffer</code>

**Description:** The data in the buffer, as a typed array

</article>


---------------------------------------

<article>

### index 
**Type:** <code>boolean</code>

**Description:** Flags whether this is an index buffer.
Index buffers are of type `ELEMENT_ARRAY_BUFFER`. Note that setting this property to false will make the buffer of type `ARRAY_BUFFER`.

For backwards compatibility.

</article>


---------------------------------------

<article>

### type 
**Type:** <code>BUFFER_TYPE</code>

**Description:** The type of buffer this is, one of&#58;

* ELEMENT_ARRAY_BUFFER - used as an index buffer
* ARRAY_BUFFER - used as an attribute buffer
* UNIFORM_BUFFER - used as a uniform buffer (if available)

</article>


---------------------------------------
