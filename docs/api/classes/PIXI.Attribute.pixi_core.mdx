[@pixi/core](../modules/pixi_core).Attribute

Holds the information for a single attribute structure required to render geometry.
This does not contain the actual data, but instead has a buffer id that maps to a [PIXI.Buffer](../classes/PIXI.Buffer.pixi_core) This can include anything from positions, uvs, normals, colors etc.

#### Parameters:
| Name       | Type                    | Default                       | Description                                                                                                            |
| ---------- | ----------------------- | ----------------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| buffer     | <code>number</code>     | <code></code>                 | the id of the buffer that this attribute will look for                                                                 |
| size       | <code>number</code>     | <code>0</code>                | the size of the attribute. If you have 2 floats per vertex (eg position x and y) this would be 2.                      |
| normalized | <code>boolean</code>    | <code>false</code>            | should the data be normalized.                                                                                         |
| type       | <code>PIXI.TYPES</code> | <code>PIXI.TYPES.FLOAT</code> | what type of number is the attribute. Check [PIXI.TYPES](../enums/PIXI.TYPES.pixi_constants) to see the ones available |
| stride     | <code>number</code>     | <code>0</code>                | How far apart, in bytes, the start of each value is. (used for interleaving data)                                      |
| start      | <code>number</code>     | <code>0</code>                | How far into the array to start reading values (used for interleaving data)                                            |
| instance   | <code>boolean</code>    | <code>false</code>            | Whether the geometry is instanced.                                                                                     |
| divisor    | <code>number</code>     | <code>1</code>                | Divisor to use when doing instanced rendering                                                                          |
