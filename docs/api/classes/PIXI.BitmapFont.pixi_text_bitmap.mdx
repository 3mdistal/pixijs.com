[@pixi/text-bitmap](../modules/pixi_text_bitmap).BitmapFont

BitmapFont represents a typeface available for use with the BitmapText class. Use the `install`method for adding a font to be used.

| ------------ | --------------------------------------------------------- | ------------- | -------------------------------------------------------------------------- |

## Methods
<article>

### from 
**Type:** <code>(name&#58; string, textStyle&#58; TextStyle &#124; Partial&#60;ITextStyle&#62;, options&#58; IBitmapFontOptions) -&#62; BitmapFont</code>

**Description:** Generates a bitmap-font for the given style and character set. This does not support kernings yet. With `style` properties, only the following non-layout properties are used&#58;

* [dropShadow](../classes/PIXI.TextStyle.pixi_text#dropShadow)
* [dropShadowDistance](../classes/PIXI.TextStyle.pixi_text#dropShadowDistance)
* [dropShadowColor](../classes/PIXI.TextStyle.pixi_text#dropShadowColor)
* [dropShadowBlur](../classes/PIXI.TextStyle.pixi_text#dropShadowBlur)
* [dropShadowAngle](../classes/PIXI.TextStyle.pixi_text#dropShadowAngle)
* [fill](../classes/PIXI.TextStyle.pixi_text#fill)
* [fillGradientStops](../classes/PIXI.TextStyle.pixi_text#fillGradientStops)
* [fillGradientType](../classes/PIXI.TextStyle.pixi_text#fillGradientType)
* [fontFamily](../classes/PIXI.TextStyle.pixi_text#fontFamily)
* [fontSize](../classes/PIXI.TextStyle.pixi_text#fontSize)
* [fontVariant](../classes/PIXI.TextStyle.pixi_text#fontVariant)
* [fontWeight](../classes/PIXI.TextStyle.pixi_text#fontWeight)
* [lineJoin](../classes/PIXI.TextStyle.pixi_text#lineJoin)
* [miterLimit](../classes/PIXI.TextStyle.pixi_text#miterLimit)
* [stroke](../classes/PIXI.TextStyle.pixi_text#stroke)
* [strokeThickness](../classes/PIXI.TextStyle.pixi_text#strokeThickness)
* [textBaseline](../classes/PIXI.TextStyle.pixi_text#textBaseline)


```js
 import { BitmapFont, BitmapText } from 'pixi.js';

 BitmapFont.from('TitleFont', {
     fontFamily: 'Arial',
     fontSize: 12,
     strokeThickness: 2,
     fill: 'purple',
 });

 const title = new BitmapText('This is the title', { fontName: 'TitleFont' });
```
      

#### Parameters:
| Name      | Type                                                      | Description                                         |
| --------- | --------------------------------------------------------- | --------------------------------------------------- |
| name      | <code>string</code>                                       | The name of the custom font to use with BitmapText. |
| textStyle | <code>TextStyle &#124; Partial&#60;ITextStyle&#62;</code> | Style options to render with BitmapFont.            |
| options   | <code>IBitmapFontOptions</code>                           | Setup options for font or name of the font.         |
#### Returns:
| Type                    | Description                      |
| ----------------------- | -------------------------------- |
| <code>BitmapFont</code> | Font generated by style options. |
</article>

### install 
**Type:** <code>(data&#58; string &#124; XMLDocument &#124; BitmapFontData, textures&#58; Texture &#124; Texture[] &#124; utils.Dict&#60;Texture&#62;, ownsTextures&#58; boolean) -&#62; PIXI.BitmapFont</code>

**Description:** Register a new bitmap font.

#### Parameters:
| Name         | Type                                                                     | Description                                                                                                                                                    |
| ------------ | ------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| data         | <code>string &#124; XMLDocument &#124; BitmapFontData</code>             | The characters map that could be provided as xml or raw string.                                                                                                |
| textures     | <code>Texture &#124; Texture[] &#124; utils.Dict&#60;Texture&#62;</code> | List of textures for each page.                                                                                                                                |
| ownsTextures | <code>boolean</code>                                                     | Set to `true` to destroy page textures when the font is uninstalled. By default fonts created with`BitmapFont.from` or from the `BitmapFontLoader` are `true`. |
#### Returns:
| Type                         | Description                                                     |
| ---------------------------- | --------------------------------------------------------------- |
| <code>PIXI.BitmapFont</code> | Result font object with font, size, lineHeight and char fields. |
</article>
