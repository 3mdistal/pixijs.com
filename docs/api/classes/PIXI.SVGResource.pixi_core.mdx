### new PIXI.SVGResource: <span><code>(sourceBase64&#58; string, options&#58; object) </code></span>

#### Parameters:
| Name             | Type                 | Default           | Description                                                              |
| ---------------- | -------------------- | ----------------- | ------------------------------------------------------------------------ |
| sourceBase64     | <code>string</code>  | <code></code>     | Base64 encoded SVG element or URL for SVG file.                          |
| options          | <code>object</code>  | <code></code>     | Options to use                                                           |
| options.scale    | <code>number</code>  | <code>1</code>    | Scale to apply to SVG. Overridden by...                                  |
| options.width    | <code>number</code>  | <code></code>     | Rasterize SVG this wide. Aspect ratio preserved if height not specified. |
| options.height   | <code>number</code>  | <code></code>     | Rasterize SVG this high. Aspect ratio preserved if width not specified.  |
| options.autoLoad | <code>boolean</code> | <code>true</code> | Start loading right away.                                                |
