[@pixi/core](../modules/pixi_core).RenderTexturePool

Texture pool, used by FilterSystem and plugins.
Stores collection of temporary pow2 or screen-sized renderTextures

If you use custom RenderTexturePool for your filters, you can use methods`getFilterTexture` and `returnFilterTexture` same as in

## Members
<article>

### SCREEN_KEY: <span><code>number</code></span>

</article>

<article>

### enableFullScreen: <span><code>boolean</code></span>
- **Default Value**: <code>false</code>

Allow renderTextures of the same size as screen, not just pow2
Automatically sets to true after `setScreenSize`

### getFilterTexture: <span><code>(input&#58; RenderTexture, resolution&#58; number, multisample&#58; MSAA_QUALITY) -&#62; RenderTexture</code></span>
Gets extra texture of the same size as input renderTexture
`getFilterTexture(input, 0.5)` or `getFilterTexture(0.5, input)`

#### Parameters:
| Name        | Type                       | Description                                                                 |
| ----------- | -------------------------- | --------------------------------------------------------------------------- |
| input       | <code>RenderTexture</code> | renderTexture from which size and resolution will be copied                 |
| resolution  | <code>number</code>        | override resolution of the renderTexture It overrides, it does not multiply |
| multisample | <code>MSAA_QUALITY</code>  | number of samples of the renderTexture                                      |
#### Returns:
| Type                       | Description |
| -------------------------- | ----------- |
| <code>RenderTexture</code> |             |

<article>

### setScreenSize: <span><code>(size&#58; ISize) -&#62; void</code></span>
If screen size was changed, drops all screen-sized textures, sets new screen size, sets `enableFullScreen` to true

Size is measured in pixels, `renderer.view` can be passed here, not `renderer.screen`

#### Parameters:
| Name | Type               | Description             |
| ---- | ------------------ | ----------------------- |
| size | <code>ISize</code> | Initial size of screen. |
</article>
