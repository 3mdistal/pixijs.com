[@pixi/prepare](../modules/pixi_prepare).BasePrepare

</div>

The prepare manager provides functionality to upload content to the GPU.
BasePrepare handles basic queuing functionality and is extended by [PIXI.Prepare](../classes/PIXI.Prepare.pixi_prepare) and [PIXI.CanvasPrepare](../classes/PIXI.CanvasPrepare.pixi_canvas_prepare) to provide preparation capabilities specific to their respective renderers.


```js
 // Create a sprite
 const sprite = PIXI.Sprite.from('something.png');

 // Load object into GPU
 app.renderer.prepare.upload(sprite, () => {
     // Texture(s) has been uploaded to GPU
     app.stage.addChild(sprite);
 });
```
