[@pixi/math](../modules/pixi_math).RoundedRectangle


The Rounded Rectangle object is an area that has nice rounded corners, as indicated by its top-left corner point (x, y) and by its width and its height and its radius.

### new PIXI.RoundedRectangle: <span><code>(x&#58; number, y&#58; number, width&#58; number, height&#58; number, radius&#58; number) </code></span>

#### Parameters:
| Name   | Type                | Default         | Description                                                        |
| ------ | ------------------- | --------------- | ------------------------------------------------------------------ |
| x      | <code>number</code> | <code>0</code>  | The X coordinate of the upper-left corner of the rounded rectangle |
| y      | <code>number</code> | <code>0</code>  | The Y coordinate of the upper-left corner of the rounded rectangle |
| width  | <code>number</code> | <code>0</code>  | The overall width of this rounded rectangle                        |
| height | <code>number</code> | <code>0</code>  | The overall height of this rounded rectangle                       |
| radius | <code>number</code> | <code>20</code> | Controls the radius of the rounded corners                         |
