The BlurFilterPass applies a horizontal or vertical Gaussian blur to an object.



        

<article>

### Constructor:
<h4><code>new PIXI.BlurFilterPass(horizontal&#58; boolean, strength&#58; number, quality&#58; number, resolution&#58; , kernelSize&#58; number) </code></h4>

#### Parameters:
| Name       | Type                 | Default        | Description                                                         |
| ---------- | -------------------- | -------------- | ------------------------------------------------------------------- |
| horizontal | <code>boolean</code> | <code></code>  | Do pass along the x-axis (`true`) or y-axis (`false`).              |
| strength   | <code>number</code>  | <code>8</code> | The strength of the blur filter.                                    |
| quality    | <code>number</code>  | <code>4</code> | The quality of the blur filter.                                     |
| resolution | <code></code>        | <code></code>  | The resolution of the blur filter.                                  |
| kernelSize | <code>number</code>  | <code>5</code> | The kernelSize of the blur filter.Options&#58; 5, 7, 9, 11, 13, 15. |

</article>

**Type:** <code>number</code>

### quality 
**Type:** <code>number</code>

**Default Value**: <code>4</code>

**Description:** Sets the quality of the blur by modifying the number of passes. More passes means higher quality bluring but the lower the performance.

</article>


---------------------------------------

<article>

### resolution 
**Type:** <code>number</code>

**Description:** The resolution of the filter. Setting this to be lower will lower the quality but increase the performance of the filter.
