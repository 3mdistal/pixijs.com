## Members
<article>

### program 
**Type:** <code>WebGLProgram</code>

**Description:** The shader program.

</article>


---------------------------------------

<article>

### uniformBufferBindings 
**Type:** <code>Dict&#60;any&#62;</code>

**Description:** A hash that stores where UBOs are bound to on the program.

</article>


---------------------------------------

<article>

### uniformData 
**Type:** <code>Dict&#60;any&#62;</code>

**Description:** Holds the uniform data which contains uniform locations and current uniform values used for caching and preventing unneeded GPU commands.

</article>


---------------------------------------

<article>

### uniformDirtyGroups 
**Type:** <code>Dict&#60;any&#62;</code>

**Description:** A place where dirty ticks are stored for groups If a tick here does not match with the Higher level Programs tick, it means we should re upload the data.

</article>


---------------------------------------

<article>

### uniformGroups 
**Type:** <code>Dict&#60;any&#62;</code>

**Description:** UniformGroups holds the various upload functions for the shader. Each uniform group and program have a unique upload function generated.

</article>


---------------------------------------

<article>

### uniformSync 
**Type:** <code>Dict&#60;any&#62;</code>

**Description:** A hash for lazily-generated uniform uploading functions.

</article>
