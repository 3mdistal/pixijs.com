<div class="package-name">

[@pixi/core](../modules/pixi_core).BaseTexture

</div>


A Texture stores the information that represents an image. All textures have a base texture, which contains information about the source. Therefore you can have many textures all using a single BaseTexture



        

<article>

### addToCache 
**Type:** <code>(baseTexture&#58; PIXI.BaseTexture, id&#58; string) -&#62; void</code>

**Description:** Adds a BaseTexture to the global BaseTextureCache. This cache is shared across the whole PIXI object.

#### Parameters:
| Name        | Type                          | Description                                         |
| ----------- | ----------------------------- | --------------------------------------------------- |
| baseTexture | <code>PIXI.BaseTexture</code> | The BaseTexture to add to the cache.                |
| id          | <code>string</code>           | The id that the BaseTexture will be stored against. |
</article>


---------------------------------------

<article>

### from 
**Type:** <code>(source&#58; HTMLImageElement &#124; HTMLVideoElement &#124; ImageBitmap &#124; PIXI.ICanvas &#124; string &#124; Array&#60;string&#62;, options&#58; IBaseTextureOptions&#60;RO&#62;, strict&#58; boolean) -&#62; PIXI.BaseTexture</code>

**Description:** Helper function that creates a base texture based on the source you provide. The source can be - image url, image element, canvas element. If the source is an image url or an image element and not in the base texture cache, it will be created and loaded.

#### Parameters:
| Name                 | Type                                                                                                                                    | Description                                                                                                                    |
| -------------------- | --------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| source               | <code>HTMLImageElement &#124; HTMLVideoElement &#124; ImageBitmap &#124; PIXI.ICanvas &#124; string &#124; Array&#60;string&#62;</code> | The source to create base texture from.                                                                                        |
| options              | <code>IBaseTextureOptions&#60;RO&#62;</code>                                                                                            | See [PIXI.BaseTexture](../classes/PIXI.BaseTexture.pixi_core)'s constructor for options.                                       |
| options.pixiIdPrefix | <code>string</code>                                                                                                                     | If a source has no id, this is the prefix of the generated id                                                                  |
| strict               | <code>boolean</code>                                                                                                                    | Enforce strict-mode, see [PIXI.settings.STRICT_TEXTURE_CACHE](../namespaces/PIXI.settings.pixi_settings#STRICT_TEXTURE_CACHE). |
#### Returns:
| Type                          | Description           |
| ----------------------------- | --------------------- |
| <code>PIXI.BaseTexture</code> | The new base texture. |
</article>

### fromBuffer 
**Type:** <code>(buffer&#58; Float32Array &#124; Uint8Array, width&#58; number, height&#58; number, options&#58; IBaseTextureOptions) -&#62; BaseTexture&#60;BufferResource&#62;</code>

**Description:** Create a new BaseTexture with a BufferResource from a Float32Array. RGBA values are floats from 0 to 1.

#### Parameters:
| Name              | Type                                        | Description                                                                                                                                                |
| ----------------- | ------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| buffer            | <code>Float32Array &#124; Uint8Array</code> | The optional array to use, if no data is provided, a new Float32Array is created.                                                                          |
| width             | <code>number</code>                         | Width of the resource                                                                                                                                      |
| height            | <code>number</code>                         | Height of the resource                                                                                                                                     |
| options           | <code>IBaseTextureOptions</code>            | See [PIXI.BaseTexture](../classes/PIXI.BaseTexture.pixi_core)'s constructor for options. Default properties are different from the constructor's defaults. |
| options.format    | <code>PIXI.FORMATS</code>                   | GL format type                                                                                                                                             |
| options.alphaMode | <code>PIXI.ALPHA_MODES</code>               | Image alpha, not premultiplied by default                                                                                                                  |
| options.scaleMode | <code>PIXI.SCALE_MODES</code>               | Scale mode, pixelating by default                                                                                                                          |
#### Returns:
| Type                                             | Description                     |
| ------------------------------------------------ | ------------------------------- |
| <code>BaseTexture&#60;BufferResource&#62;</code> | * The resulting new BaseTexture |
</article>
