### generateTexture: <span><code>(displayObject&#58; IRenderableObject, options&#58; IGenerateTextureOptions) -&#62; RenderTexture</code></span>
A Useful function that returns a texture of the display object that can then be used to create sprites This can be quite useful if your displayObject is complicated and needs to be reused multiple times.

#### Parameters:
| Name          | Type                                 | Description                                          |
| ------------- | ------------------------------------ | ---------------------------------------------------- |
| displayObject | <code>IRenderableObject</code>       | The displayObject the object will be generated from. |
| options       | <code>IGenerateTextureOptions</code> | Generate texture options.                            |
#### Returns:
| Type                       | Description                                         |
| -------------------------- | --------------------------------------------------- |
| <code>RenderTexture</code> | a shiny new texture of the display object passed in |
