CountLimiter limits the number of items handled by a [PIXI.BasePrepare](../classes/PIXI.BasePrepare.pixi_prepare) to a specified number of items per frame.



        

<article>

### Constructor:
<h4><code>new PIXI.CountLimiter(maxItemsPerFrame&#58; number) </code></h4>

#### Parameters:
| Name             | Type                | Default       | Description                                                  |
| ---------------- | ------------------- | ------------- | ------------------------------------------------------------ |
| maxItemsPerFrame | <code>number</code> | <code></code> | The maximum number of items that can be prepared each frame. |

</article>
