System plugin to the renderer to manage geometry.



        

<article>

### Constructor:
<h4><code>new PIXI.GeometrySystem(renderer&#58; Renderer) </code></h4>

#### Parameters:
| Name     | Type                  | Default       | Description                         |
| -------- | --------------------- | ------------- | ----------------------------------- |
| renderer | <code>Renderer</code> | <code></code> | The renderer this System works for. |

</article>

---------------------------------------




## Members
<article>

### canUseUInt32ElementIndex 
**Type:** <code>boolean</code>

**Description:** `true` if support `gl.UNSIGNED_INT` in `gl.drawElements` or `gl.drawElementsInstanced`.

</article>


---------------------------------------

<article>

### hasInstance 
**Type:** <code>boolean</code>

**Description:** `true` if has `ANGLE_instanced_arrays` extension.

</article>


---------------------------------------

<article>

### hasVao 
**Type:** <code>boolean</code>

**Description:** `true` if we has `*_vertex_array_object` extension.

</article>


---------------------------------------

<article>

### managedGeometries 
**Type:** <code>&#123; [key&#58; number]&#58; Geometry &#125;</code>

**Description:** Cache for all geometries by id, used in case renderer gets destroyed or for profiling.

</article>


---------------------------------------


## Methods
<article>

### bind 
**Type:** <code>(geometry&#58; Geometry, shader&#58; Shader) -&#62; void</code>

**Description:** Binds geometry so that is can be drawn. Creating a Vao if required

#### Parameters:
| Name     | Type                  | Description                        |
| -------- | --------------------- | ---------------------------------- |
| geometry | <code>Geometry</code> | Instance of geometry to bind.      |
| shader   | <code>Shader</code>   | Instance of shader to use vao for. |
</article>


---------------------------------------

<article>

### disposeAll 
**Type:** <code>(contextLost&#58; boolean) -&#62; void</code>

**Description:** Dispose all WebGL resources of all managed geometries.

#### Parameters:
| Name        | Type                 | Description                                        |
| ----------- | -------------------- | -------------------------------------------------- |
| contextLost | <code>boolean</code> | If context was lost, we suppress `gl.delete` calls |
</article>


---------------------------------------

<article>

### disposeGeometry 
**Type:** <code>(geometry&#58; Geometry, contextLost&#58; boolean) -&#62; void</code>

**Description:** Disposes geometry.

#### Parameters:
| Name        | Type                  | Description                                        |
| ----------- | --------------------- | -------------------------------------------------- |
| geometry    | <code>Geometry</code> | Geometry with buffers. Only VAO will be disposed   |
| contextLost | <code>boolean</code>  | If context was lost, we suppress deleteVertexArray |
</article>


---------------------------------------

<article>

### draw 
**Type:** <code>(type&#58; DRAW_MODES, size&#58; number, start&#58; number, instanceCount&#58; number) -&#62; this</code>

**Description:** Draws the currently bound geometry.

#### Parameters:
| Name          | Type                    | Description                                                                                                            |
| ------------- | ----------------------- | ---------------------------------------------------------------------------------------------------------------------- |
| type          | <code>DRAW_MODES</code> | The type primitive to render.                                                                                          |
| size          | <code>number</code>     | The number of elements to be rendered. If not specified, all vertices after the starting vertex will be drawn.         |
| start         | <code>number</code>     | The starting vertex in the geometry to start drawing from. If not specified, drawing will start from the first vertex. |
| instanceCount | <code>number</code>     | The number of instances of the set of elements to execute. If not specified, all instances will be drawn.              |
#### Returns:
| Type              | Description |
| ----------------- | ----------- |
| <code>this</code> |             |
</article>

<article>

### initGeometryVao 
**Type:** <code>(geometry&#58; Geometry, shader&#58; Shader, incRefCount&#58; boolean) -&#62; WebGLVertexArrayObject</code>

**Description:** Creates or gets Vao with the same structure as the geometry and stores it on the geometry. If vao is created, it is bound automatically. We use a shader to infer what and how to set up the attribute locations.

#### Parameters:
| Name        | Type                  | Description                                  |
| ----------- | --------------------- | -------------------------------------------- |
| geometry    | <code>Geometry</code> | Instance of geometry to to generate Vao for. |
| shader      | <code>Shader</code>   | Instance of the shader.                      |
| incRefCount | <code>boolean</code>  | Increment refCount of all geometry buffers.  |
#### Returns:
| Type                                | Description |
| ----------------------------------- | ----------- |
| <code>WebGLVertexArrayObject</code> |             |
</article>
