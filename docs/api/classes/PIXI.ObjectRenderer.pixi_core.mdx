Base for a common object renderer that can be used as a system renderer plugin.



        

<article>

### Constructor:
<h4><code>new PIXI.ObjectRenderer(renderer&#58; Renderer) </code></h4>

#### Parameters:
| Name     | Type                  | Default       | Description                          |
| -------- | --------------------- | ------------- | ------------------------------------ |
| renderer | <code>Renderer</code> | <code></code> | The renderer this manager works for. |

</article>

---------------------------------------




## Members
<article>

### renderer 
**Type:** <code>Renderer</code>

**Description:** The renderer this manager works for.

</article>


---------------------------------------


## Methods
<article>

### destroy 
**Type:** <code>() -&#62; void</code>

**Description:** Generic destruction method that frees all resources. This should be called by subclasses.

</article>


---------------------------------------

<article>

### flush 
**Type:** <code>() -&#62; void</code>

**Description:** Stub method that should be used to empty the current batch by rendering objects now.

</article>


---------------------------------------

<article>

### render 
**Type:** <code>(_object&#58; PIXI.DisplayObject) -&#62; void</code>

**Description:** Keeps the object to render. It doesn't have to be rendered immediately.

#### Parameters:
| Name    | Type                            | Description           |
| ------- | ------------------------------- | --------------------- |
| _object | <code>PIXI.DisplayObject</code> | The object to render. |
</article>


---------------------------------------

<article>

### start 
**Type:** <code>() -&#62; void</code>

**Description:** Stub method that initializes any state required before rendering starts. It is different from the `prerender`signal, which occurs every frame, in that it is called whenever an object requests _this_ renderer specifically.

</article>


---------------------------------------

<article>

### stop 
**Type:** <code>() -&#62; void</code>

**Description:** Stops the renderer. It should free up any state and become dormant.

</article>
