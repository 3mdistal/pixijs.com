<article>

### updateCheck: <span><code>(func&#58; (system&#58; this, state&#58; State) =&#62; void, value&#58; boolean) -&#62; void</code></span>
Checks to see which updates should be checked based on which settings have been activated.
For example, if blend is enabled then we should check the blend modes each time the state is changed or if polygon fill is activated then we need to check if the polygon offset changes. The idea is that we only check what we have too.

#### Parameters:
| Name  | Type                                                          | Description                                    |
| ----- | ------------------------------------------------------------- | ---------------------------------------------- |
| func  | <code>(system&#58; this, state&#58; State) =&#62; void</code> | the checking function to add or remove         |
| value | <code>boolean</code>                                          | should the check function be added or removed. |
</article>
