A resource that contains a number of sources.



        

<article>

### Constructor:
<h4><code>new PIXI.ArrayResource(source&#58; number &#124; Array&#60;any&#62;, options&#58; ISize) </code></h4>

#### Parameters:
| Name           | Type                                          | Default       | Description                                                                                                          |
| -------------- | --------------------------------------------- | ------------- | -------------------------------------------------------------------------------------------------------------------- |
| source         | <code>number &#124; Array&#60;any&#62;</code> | <code></code> | Number of items in array or the collection of image URLs to use. Can also be resources, image elements, canvas, etc. |
| options        | <code>ISize</code>                            | <code></code> | Options to apply to PIXI.autoDetectResource                                                                          |
| options.width  | <code>number</code>                           | <code></code> | Width of the resource                                                                                                |
| options.height | <code>number</code>                           | <code></code> | Height of the resource                                                                                               |

</article>
