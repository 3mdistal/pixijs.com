Slightly opinionated default shader for PixiJS 2D objects.



        

<article>

### Constructor:
<h4><code>new PIXI.MeshMaterial(uSampler&#58; Texture, options&#58; IMeshMaterialOptions) </code></h4>

#### Parameters:
| Name               | Type                              | Default               | Description                           |
| ------------------ | --------------------------------- | --------------------- | ------------------------------------- |
| uSampler           | <code>Texture</code>              | <code></code>         | Texture that material uses to render. |
| options            | <code>IMeshMaterialOptions</code> | <code></code>         | Additional options                    |
| options.alpha      | <code>number</code>               | <code>1</code>        | Default alpha.                        |
| options.tint       | <code>PIXI.ColorSource</code>     | <code>0xFFFFFF</code> | Default tint.                         |
| options.pluginName | <code>string</code>               | <code>'batch'</code>  | Renderer plugin for batching.         |
| options.program    | <code>PIXI.Program</code>         | <code>0xFFFFFF</code> | Custom program.                       |
| options.uniforms   | <code>object</code>               | <code></code>         | Custom uniforms.                      |

</article>

## Members
<article>

### alpha 
**Type:** <code>number</code>

**Default Value**: <code>1</code>

**Description:** This gets automatically set by the object using this.

---------------------------------------

<article>

### batchable 
**Type:** <code>boolean</code>

**Default Value**: <code>true</code>

**Description:** `true` if shader can be batch with the renderer's batch system.

</article>


---------------------------------------

<article>

### pluginName 
**Type:** <code>string</code>

**Default Value**: <code>'batch'</code>

**Description:** Renderer plugin for batching.

</article>


---------------------------------------

<article>

### program 
**Type:** <code>Program</code>

**Description:** Program that the shader uses.

</article>


---------------------------------------

<article>

### texture 
**Type:** <code>Texture</code>

**Description:** Reference to the texture being rendered.

</article>


---------------------------------------

<article>

### tint 
**Type:** <code>ColorSource</code>

**Default Value**: <code>0xFFFFFF</code>

</article>


---------------------------------------

<article>

### uniforms
