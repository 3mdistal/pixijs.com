### convertToBlob 
**Type:** <code>(options&#58; object) -&#62; Promise&#60;Blob&#62;</code>

**Description:** Get the content of the canvas as Blob.

#### Parameters:
| Name            | Type                | Description                                                                                          |
| --------------- | ------------------- | ---------------------------------------------------------------------------------------------------- |
| options         | <code>object</code> | The options for creating Blob.                                                                       |
| options.type    | <code>string</code> | The MIME type for the image format to return. If not specify, the default value is image/png.        |
| options.quality | <code>string</code> | The image quality to be used when creating images using file formats that support lossy compression. |
#### Returns:
| Type                               | Description                        |
| ---------------------------------- | ---------------------------------- |
| <code>Promise&#60;Blob&#62;</code> | The content of the canvas as Blob. |
</article>

### dispatchEvent 
**Type:** <code>(event&#58; Event) -&#62; boolean</code>

**Description:** Dispatches a event.

#### Parameters:
| Name  | Type               | Description                                                                                     |
| ----- | ------------------ | ----------------------------------------------------------------------------------------------- |
| event | <code>Event</code> | The Event object to dispatch. Its Event.target property will be set to the current EventTarget. |
#### Returns:
| Type                 | Description                                                                                                                                      |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| <code>boolean</code> | Returns false if event is cancelable, and at least one of the event handlers which received event called Event.preventDefault(). Otherwise true. |
