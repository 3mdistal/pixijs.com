Anisotropic filtering level of texture.

<article>

### chars: <span><code>string &#124; (string &#124; string[])[]</code></span>
- **Default Value**: <code>PIXI.BitmapFont.ALPHANUMERIC</code>

Characters included in the font set. You can also use ranges. For example, `[['a', 'z'], ['A', 'Z'], "!@#$%^&*()~{}[] "]`. Don't forget to include spaces ' ' in your character set!

</article>


---------------------------------------

<article>

### mipmap: <span><code>MIPMAP_MODES</code></span>
- **Default Value**: <code>PIXI.BaseTexture.defaultOptions.mipmap</code>

If mipmapping is enabled for texture. For instance, by default PixiJS only enables mipmapping on Power-of-Two textures. If your textureWidth or textureHeight are not power-of-two, you may consider enabling mipmapping to get better quality with lower font sizes. Note&#58; for MSDF/SDF fonts, mipmapping is not supported.

</article>


---------------------------------------

<article>

### padding: <span><code>number</code></span>
- **Default Value**: <code>4</code>

Padding between glyphs on texture atlas. Lower values could mean more visual artifacts and bleeding from other glyphs, larger values increase the space required on the texture.

</article>


---------------------------------------

<article>

### resolution: <span><code>number</code></span>
- **Default Value**: <code>1</code>

Render resolution for glyphs.

</article>


---------------------------------------

<article>

### scaleMode: <span><code>SCALE_MODES</code></span>
- **Default Value**: <code>PIXI.BaseTexture.defaultOptions.scaleMode</code>

Default scale mode, linear, nearest. Nearest can be helpful for bitmap-style fonts.

</article>


---------------------------------------

<article>

### textureHeight: <span><code>number</code></span>
- **Default Value**: <code>512</code>

Optional height of atlas, smaller values to reduce memory.

</article>


---------------------------------------

<article>

### textureWidth: <span><code>number</code></span>
- **Default Value**: <code>512</code>

Optional width of atlas, smaller values to reduce memory.

</article>
