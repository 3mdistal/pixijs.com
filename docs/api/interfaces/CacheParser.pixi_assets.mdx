[@pixi/assets](../modules/pixi_assets).CacheParser


For every asset that is cached, it will call the parsers test function the flow is as follows&#58;

1. `cacheParser.test()`&#58; Test the asset.
2. `cacheParser.getCacheableAssets()`&#58; If the test passes call the getCacheableAssets function with the asset

Useful if you want to add more than just a raw asset to the cache (for example a spritesheet will want to make all its sub textures easily accessible in the cache)
