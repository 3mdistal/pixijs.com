### sharedTicker 
**Type:** <code>boolean</code>

**Default Value**: <code>false</code>

**Description:** Set`true` to use `Ticker.shared`, `false` to create new ticker. If set to `false`, you cannot register a handler to occur before anything that runs on the shared ticker. The system ticker will always run before both the shared ticker and the app ticker.

</article>
