[@pixi/constants](../modules/pixi_constants).SCALE_MODES

The scale modes that are supported by pixi.
The [PIXI.BaseTexture.defaultOptions.scaleMode](../classes/PIXI.BaseTexture.pixi_core#defaultOptions) scale mode affects the default scaling mode of future operations. It can be re-assigned to either LINEAR or NEAREST, depending upon suitability.
