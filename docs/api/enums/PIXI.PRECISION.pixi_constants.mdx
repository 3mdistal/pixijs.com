[@pixi/constants](../modules/pixi_constants).PRECISION

Constants that specify float precision in shaders.



        

## Summary
| Name                | Default | Description                                                                                                                                                             |
| ------------------- | ------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <code>HIGH</code>   | highp   | highp is at least a 32 bit value. For floating point values they can range from&#58; -2^62 to +2^62, for integer values they are similar to Uint32Array or Int32Array   |
| <code>LOW</code>    | lowp    | lowp is at least an 9 bit value. For floating point values they can range from&#58; -2 to +2, for integer values they are similar to Uint8Array or Int8Array            |
| <code>MEDIUM</code> | mediump | mediump is at least a 16 bit value. For floating point values they can range from&#58; -2^14 to +2^14, for integer values they are similar to Uint16Array or Int16Array |
