<div class="package-name">

[@pixi/constants](../modules/pixi_constants).COLOR_MASK_BITS

</div>


Bitwise OR of masks that indicate the color channels that are rendered to.



        

## Summary
| Name               | Default | Description    |
| ------------------ | ------- | -------------- |
| <code>ALPHA</code> | 0x      | Alpha channel. |
| <code>BLUE</code>  | 0x4     | Blue channel.  |
| <code>GREEN</code> | 0x2     | Green channel  |
| <code>RED</code>   | 0x1     | Red channel.   |
