[@pixi/assets](../modules/pixi_assets).[PIXI](../namespaces/PIXI).LoaderParserPriority


The extension priority for loader parsers. Helpful when managing multiple parsers that share the same extension test. The higher priority parsers will be checked first.



// TODO: @see

        

## Summary
| Name                | Default | Description                                                          |
| ------------------- | ------- | -------------------------------------------------------------------- |
| <code>High</code>   |         | Specific texture types&#58; svg, png, ktx, dds, basis                |
| <code>Low</code>    |         | Generic parsers&#58; txt, json, webfonts                             |
| <code>Normal</code> |         | PixiJS assets with generic extensions&#58; spritesheets, bitmapfonts |
