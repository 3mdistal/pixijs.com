[@pixi/constants](../modules/pixi_constants).ALPHA_MODES


How to treat textures with premultiplied alpha



// TODO: @see

        

## Summary
| Name                                | Default | Description                                                                                                                  |
| ----------------------------------- | ------- | ---------------------------------------------------------------------------------------------------------------------------- |
| <code>NO_PREMULTIPLIED_ALPHA</code> | 0       | Source is not premultiplied, leave it like that. Option for compressed and data textures that are created from typed arrays. |
| <code>NPM</code>                    | 0       | Alias for NO_PREMULTIPLIED_ALPHA.                                                                                            |
| <code>PMA</code>                    | 2       | Alias for PREMULTIPLIED_ALPHA.                                                                                               |
| <code>PREMULTIPLIED_ALPHA</code>    | 2       | Source is already premultiplied. Example&#58; spine atlases with `_pma` suffix.                                              |
| <code>PREMULTIPLY_ON_UPLOAD</code>  | 1       | Source is not premultiplied, premultiply on upload. Default option, used for all loaded images.                              |
| <code>UNPACK</code>                 | 1       | Default option, alias for PREMULTIPLY_ON_UPLOAD.                                                                             |
