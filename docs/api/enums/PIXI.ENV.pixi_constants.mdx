Different types of environments for WebGL.



        

## Summary
| Name                      | Default | Description                                                                                                                                                                    |
| ------------------------- | ------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| <code>WEBGL</code>        | 1       | Version 1 of WebGL                                                                                                                                                             |
| <code>WEBGL_LEGACY</code> | 0       | Used for older v1 WebGL devices. PixiJS will aim to ensure compatibility with older / less advanced devices. If you experience unexplained flickering prefer this environment. |
| <code>WEBGL2</code>       | 2       | Version 2 of WebGL                                                                                                                                                             |
