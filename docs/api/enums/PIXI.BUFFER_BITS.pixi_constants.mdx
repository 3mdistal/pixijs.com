[@pixi/constants](../modules/pixi_constants).BUFFER_BITS

</div>


Bitwise OR of masks that indicate the buffers to be cleared.



        

## Summary
| Name                 | Default    | Description                                                |
| -------------------- | ---------- | ---------------------------------------------------------- |
| <code>COLOR</code>   | 0x00004000 | Indicates the buffers currently enabled for color writing. |
| <code>DEPTH</code>   | 0x00000100 | Indicates the depth buffer.                                |
| <code>STENCIL</code> | 0x00000400 | Indicates the stencil buffer.                              |
