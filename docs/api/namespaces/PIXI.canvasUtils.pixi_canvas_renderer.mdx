Utility methods for Sprite/Texture tinting.
Tinting with the CanvasRenderer involves creating a new canvas to use as a texture, so be aware of the performance implications.



        







## Members
<article>

### cacheStepsPerColorChannel 
**Type:** <code>number</code>

**Default Value**: <code>8</code>

**Description:** Number of steps which will be used as a cap when rounding colors.

</article>


---------------------------------------

<article>

### canUseMultiply 
**Type:** <code>boolean</code>

**Description:** Whether or not the Canvas BlendModes are supported, consequently the ability to tint using the multiply method.

</article>


---------------------------------------

<article>

### convertTintToImage 
**Type:** <code>boolean</code>

**Default Value**: <code>false</code>

**Description:** Tint cache boolean flag.

</article>


---------------------------------------

<article>

### tintMethod 
**Type:** <code>Function</code>

**Description:** The tinting method that will be used.

</article>
